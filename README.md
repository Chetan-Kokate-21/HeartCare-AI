# ❤️ HeartCare AI

An AI-powered Heart Disease Prediction System built using **React, Flask, Python, and Machine Learning**.

---

## 📌 Project Overview

HeartCare AI predicts whether a patient has a **Low Risk** or **High Risk** of heart disease based on health parameters entered by the user.

The application uses a trained **K-Nearest Neighbors (KNN)** Machine Learning model to generate predictions and confidence scores through a modern React interface.

---

## ✨ Features

- ❤️ Heart Disease Prediction
- 🤖 Machine Learning (KNN)
- 📊 Prediction Confidence Score
- 💚 Heart Health Score
- ⚠️ Risk Factor Analysis
- 📱 Responsive Modern UI
- ⚡ Fast Predictions
- 🎨 Beautiful React + Tailwind Interface
